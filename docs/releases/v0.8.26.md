## 重要：v0.8.25 升级后 reboot 仍离线请读本节

`Required info is missing. Outdated version?` 表示 **rw-core 未在线**（healthcheck 中 `xrayInternalStatusCached=false`）。

### 常见原因

1. **从未在 v0.8.24+ 成功 xray/start**：`/var/lib/remnanode/last-start.json` 不存在，reboot 后无法自动恢复。
2. **RestoreOnBoot 只尝试一次**：慢启动 VPS 上 gRPC 未就绪即失败（v0.8.26 已加重试）。

### v0.8.26 修复

- RestoreOnBoot **10 次重试**（间隔 2s，对齐官方 pRetry）
- 进程 SIGTERM 关机时 **flush last-start.json**（内存中有配置时）
- `remnanode-lite doctor` 检查 `last-start.json` 是否存在

### 升级与验证

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v0.8.26/scripts/upgrade.sh | sudo bash -s -- --yes
sudo systemctl restart remnawave-node

# 1) 自检
sudo remnanode-lite doctor

# 2) Panel 禁用 → 启用节点一次（生成 last-start.json）
ls -la /var/lib/remnanode/last-start.json

# 3) 重启验证
sudo reboot
```
