## 重要：请从 ≤0.8.21 升级

修复 **xray/start 启动死锁**：rw-core 能启动、`:61000` 能监听，但 lite-go 在 gRPC Ping 阶段因 `xrayOnline` 门控永远失败，约 20s 后误杀 rw-core，Panel 显示 `Required info is missing. Outdated version?`。

**0.8.22 已在 KDDI Alpine 256MB 生产验证通过。**

### 修复

- **gRPC Ping 启动死锁**：`PingXrayGRPC` 在 rw-core 刚启动、尚未标记 online 时不再被 `statsAPI` 拒绝。
- **菜单升级**：安装脚本菜单选「升级」时自动 `--yes`，避免无 TTY 环境下升级半途取消。

### 升级

```bash
# Alpine
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v0.8.22/scripts/install-node-alpine.sh -o /tmp/install-alpine.sh
bash /tmp/install-alpine.sh --upgrade --yes
rc-service remnawave-node restart

# systemd
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v0.8.22/scripts/upgrade.sh | sudo bash -s -- --yes
sudo systemctl restart remnawave-node
```

升级后在 Panel 对节点 **禁用 → 启用** 一次。

完整变更见 [CHANGELOG.md](../CHANGELOG.md).
