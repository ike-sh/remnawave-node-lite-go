## v1.0.0 — 稳定正式版

首个生产就绪版本。Panel 2.7.x 主流程已在真实 VPS 验证：全新安装、原地升级、reboot 自动恢复。

### 亮点

- 28 条 REST API 与官方 `@remnawave/node` v2.7.0 对齐
- Go 单二进制，内存占用约为官方 Node.js 版的 1/5
- 一键安装 / 升级 / 卸载（systemd + OpenRC）
- 首次安装 Panel 自动上线（10s 健康检查）
- 可选：`CUSTOM_CORE_URL`、geo-zapret、低内存模式

### 升级

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v1.0.0/scripts/upgrade.sh | sudo bash -s -- --yes
sudo systemctl restart remnawave-node
```

### 全新安装

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v1.0.0/scripts/install-node.sh | sudo bash
```

与 v0.8.30 代码等价，主要变更为文档整理与版本号正式化。
