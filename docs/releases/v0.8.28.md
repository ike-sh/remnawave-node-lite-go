## 重要：修复首次安装后 Panel 不上线

v0.8.27 及更早版本在节点 HTTPS 已监听、rw-core 尚未启动时，`GET /node/stats/get-system-stats` 返回 `200 + xrayInfo:null`。Remnawave Panel 健康检查（每 10s）将此视为「已连接但 xray 异常」，**不会**入队 `startNode`，导致需手动 **禁用→启用** 才能上线。

**v0.8.28** 对齐官方 `@remnawave/node`：rw-core 离线时返回 `500 A010`，Panel 走 `handleDisconnectedNode` 并自动触发 `startNode`。

### 升级

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v0.8.28/scripts/upgrade.sh | sudo bash -s -- --yes
sudo systemctl restart remnawave-node
```

### 验证（新装或升级后）

1. Panel 启用节点后等待约 **10～20 秒**（无需手动切换）
2. `journalctl -u remnawave-node -f` 应出现 `xray/start succeeded`
3. `ls /var/lib/remnanode/last-start.json` 应存在
