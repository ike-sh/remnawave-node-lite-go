# v1.0.2 — 审计 M2–M6

## 摘要

延续 v1.0.1 安全加固，处理审计报告中 M2–M6 项。

## 变更

| 项 | 处理 |
|----|------|
| M2 | JWT `iss`/`aud`/`sub` 存在时校验 Panel 约定值 |
| M3 | Vision `block-ip` / `unblock-ip` IP 格式校验 |
| M4 | 新增 `NODE_BIND_ADDR` 可选绑定地址 |
| M5 | 默认 body limit 1000MB → 256MB |
| M6 | 内部 token 未配置时拒绝 Unix API |

## 升级

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v1.0.2/scripts/upgrade.sh | sudo bash -s -- --yes
```

大节点若需更大请求体，在 `node.env` 设置 `BODY_LIMIT_MB=1000`。
