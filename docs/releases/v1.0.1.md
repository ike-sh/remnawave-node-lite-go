# v1.0.1 — 安全加固

## 摘要

针对静态审计 H1–M1 项的安全修复，**无 Panel 契约变更**（GET `/node/xray/stop` 仍可用）。

## 变更

- 内部 REST Token 不再写入 rw-core 进程参数或 webhook URL
- Unix socket `internal.sock` 创建后 `chmod 0600`
- zstd 请求：压缩体 ≤64MB，解压后受 body limit 约束
- `/node/xray/stop` 支持 POST（推荐）；GET 保留并记弃用日志

## 升级

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v1.0.1/scripts/upgrade.sh | sudo bash -s -- --yes
sudo remnanode-lite doctor
```

## 兼容性

- 旧版 rw-core 若仍带 `?token=` 查询参数，服务端继续接受
- 升级后 rw-core 将自动以无 token URL 重启（需 Panel 触发 start 或重启服务）
