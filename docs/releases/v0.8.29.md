## v0.8.29 — Panel 2.7.x 功能对齐补齐

### 新增

- **`CUSTOM_CORE_URL`**：在 `node.env` 或环境变量中设置后，`install-xray.sh` 从自定义 URL 下载 rw-core（行为对齐官方 Docker entrypoint）。
- **zapret geo 文件**：`GEO_ZAPRET_FILE` / `IP_ZAPRET_FILE` 指向本地文件，安装时复制到 `$GEO_DIR`（`geo-zapret.dat` / `ip-zapret.dat`），供 Xray `ext:` 路由引用。

### 修复

- gRPC 就绪检测间隔改为 **2s**（对齐官方 `pRetry`）。
- Stats IP 类接口 gRPC 失败时返回 **200 + 空数据**（对齐官方 node）。

### 升级

```bash
curl -fsSL https://raw.githubusercontent.com/ike-sh/remnawave-node-lite-go/v0.8.29/scripts/upgrade.sh | sudo bash -s -- --yes
sudo systemctl restart remnawave-node
```

### 配置示例（`node.env`）

```env
CUSTOM_CORE_URL=https://example.com/my-xray-build
GEO_ZAPRET_FILE=/opt/rules/geo-zapret.dat
IP_ZAPRET_FILE=/opt/rules/ip-zapret.dat
```
